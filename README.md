# 🏛️ LuxEstate

> An ultra-premium, modern, and immersive Real Estate web application.

LuxEstate is designed to provide a "glory" experience for high-end property browsing. It features a cinematic user experience, glassmorphism UI elements, smooth framer-motion animations, and full RTL (Right-to-Left) language compatibility for a truly global audience. 

## ✨ Key Features

- **Ultra-Premium Design:** Built with a sophisticated color palette (Deep surface blacks `#050505` and custom primary gold accents).
- **Cinematic Animations:** Powered by `framer-motion` for fluid page transitions, sophisticated hover effects, and spring-physics element interactions.
- **Glassmorphism UI:** Utilizes frosted glass panels, dynamic blur backgrounds, atmospheric gradients, and elegant border glows.
- **RTL & LTR Support:** Fully localized experience seamlessly adapting layouts, typography, and icons based on language direction.
- **Responsive Architecture:** Pixel-perfect adjustments from massive wide desktop displays down to mobile screens.
- **Modern Tech Stack:** React, TypeScript, Vite, and Tailwind CSS.


## 🛠️ Technology Stack

- **Framework:** React (Vite)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Routing:** React Router v6
